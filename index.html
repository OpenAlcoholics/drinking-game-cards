drinking-game-cards

object
text: string (at least 1 chars)

The task text. It may contain {int} as a placeholder for a random number.

count: integer , { x ∈ ℤ | x ≥ 1 }

How many times the card can occur in one session -> implicit higher probability

uses: integer , { x ∈ ℤ | x ≥ 0 }

How often the (interactive) card can be used.

rounds: integer , { x ∈ ℤ | x ≥ -1 }

How many rounds the card is active for. If uses > 1, the rounds are counted after each use. Use -1 for infinite duration.

id: integer

The ID of this task. Note that this is NOT always the index in the tasks list.

personal: boolean

Whether the task is for a specific player or the whole group.

remote: boolean

Whether this task can be used in a remote game (i.e. not all players are in the same room).

unique: boolean

Whether only one instance of this task can be active (rounds != 0 && count > 1). The latest card replaces previous ones.
